# appleLike-Slide

## 预览
[点击这里预览](https://strugglebak.github.io/appleLike-Slide/)

## 功能
一个 Apple 风格的轮播 Demo,主要实现的功能有

- 定时播放，这里设定的时间是 2s
- 鼠标移入轮播主界面时，轮播停止;鼠标移出轮播主界面时，继续轮播
- 底下的 menu item 点击可以跳转到相应图片显示
- 点击底部的 menu item 时,鼠标的状态会有变化
- inactive 状态的 menu item,若有鼠标悬停，背景会变白
